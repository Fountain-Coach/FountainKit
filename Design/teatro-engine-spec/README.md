# Teatro Engine Spec · Draft v0.1

This package contains the initial formal specification for the **Teatro Engine**:

- `style/teatro-style-spec.md` — visual and interaction style grammar (Teatro look & feel).
- `openapi/teatro-engine.yaml` — OpenAPI 3.1 draft for controlling Teatro scenes, props, and marionette rigs.
- `notes/roadmap.md` — short notes on how to evolve the spec further.

The engine is designed for:

- Orthographic isometric 3D stages
- Teatro-style black-on-paper visuals
- Systemic / constellation work and marionette (Fadenpuppe) rigs
