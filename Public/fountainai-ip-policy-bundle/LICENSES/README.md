# FountainAI IP & Licensing Overview

This directory defines the canonical intellectual property framework for the FountainCoach and FountainAI projects.
It harmonizes licensing, authorship, and attribution across all repositories in the organization.

## Overview
FountainAI operates under a dual-layer licensing model:
- Code and SDKs — Apache 2.0 License
- Creative and Generated Works — Creative Commons Attribution–NonCommercial 4.0

AI-generated content is always attributed and co-owned under the FountainCoach organization.
Provenance is embedded in each artifact through automated metadata tagging and commit annotations.

For full details, refer to `.fountain/ip-policy.yml` at the repository root.
